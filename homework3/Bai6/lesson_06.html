5. Group of selector

A comma-separated list of selector represents the union of all elements selected by each of the individual selectors in the list. (A cooma is U+002C.) For examle, in CSS when several selector share the same declaration, they may be grouped into a comma-separated list. White space may appear before and/or after the comma.

CSS examle

In this example, we condense three rules with indentical declarations into one. Thus,

h1 { font-family : sans-serif}

h2 { font-family : sans-serif}

h3 { font-family : sans-serif}

is equivalent to:

h1, h2 ,h3 { font-family: sans-serif}
